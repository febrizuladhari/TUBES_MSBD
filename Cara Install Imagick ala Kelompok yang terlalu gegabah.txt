Tutorial Install Imagick ala kelompok yang terlalu gegabah

pertama-tama buka xampp run apache dan mysql

https://imagemagick.org/script/download.php#windows
download dulu imagick nya di link atas

install next-next aja

setelah diinstal, buka cmd. Run perintah magick -version
kalau udah keluar versi nya berarti udh diinstal tapi belum bisa diimplementasikan


buka http://localhost/dashboard/phpinfo.php

cek php version paling atas : diingat
lalu Architecture : itu ada x64 atau x86
lalu cari Thread Safety, ada yang enabled atau disabled. beda" tiap device

download file dll untuk ekstensi nya dibawah. sesuaikan dengan versi php yg udah dicek dan
thred safety nya serta architecture nya 
https://mlocati.github.io/articles/php-windows-imagick.html

ekstrak folder nya
pindahkan file php_imagick.dll ke c:\xampp\php\ext

buka php.ini dari xampp dari service apache. tambahkan code di bawah ini dimana aja
extension=php_imagick.dll


tapi belum selesai boi. Buka lagi folder yang diekstrak tadi.
select semua file yang namanya
CORE_RL_xxx.dll
IM_MOD_RL_xxx.dll
FILTER_analyze_.dll
ImageMagick.dll

copy semua file ke c:\xampp\apache\bin

restart semua service. atau disarankan restart laptop.
lalu buka cmd jalankan perintah php -m
lihat imagick udh ada atau belum
